# Simple Example of Integrated Information Theory

This page shows a simple example related to Integrated Information Theory (IIT).

## Example

This example compares two systems.

The first system has two independent sensors.  
Each sensor works separately, so the system does not have integrated information.

The second system has two connected sensors.  
The sensors work together as one system, so the system has integrated information.

## Python File

The file `iit_example.py` shows a simple bar graph.

The graph compares the integrated information value, phi, of the two systems.

## How to Run

```bash
python iit_example.py
